# Data for Career Uncertainty Drivers Project

## Data Source
This project uses the **ASPIRES3** dataset (Young People's STEM Aspirations and Trajectories, Age 20‑22, 2020‑2023).

- **Provider:** UK Data Service
- **Study Number:** SN 9224
- **Sample:** 7,635 respondents in England, born 1998‑99

## Access Instructions
Due to licensing restrictions, the raw data cannot be included in this repository. To access the data:

1. Register at [UK Data Service](https://ukdataservice.ac.uk/)
2. Apply for access to study **SN 9224** (ASPIRES3)
3. Download the data files
4. Place the following files in this directory:
   - `aspires3_data.tab` (or .csv)
   - Associated documentation files (optional)

## Data Processing
After downloading, run the `scripts/data_cleaning.py` script to generate the cleaned dataset used in analysis.
